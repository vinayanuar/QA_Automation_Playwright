# Playwright QA Automation Project

This project demonstrates End-to-End (E2E) UI automation and API automation using Playwright.

The automation covers:
- UI testing with Page Object Model (POM)
- Complete checkout flow
- API testing (GET & POST)
- Screenshot & video capture on failure
- Test grouping using test.describe

---

##  Tech Stack
- Playwright
- JavaScript (Node.js)
- Page Object Model (POM)
